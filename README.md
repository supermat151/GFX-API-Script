# GFX Long Duration Automation Script

Automates graphics benchmark testing for extended durations with comprehensive logging and monitoring.

## Purpose

This script runs graphics benchmarks (3DMark, Unigine, UL Procyon) for specified time periods to test GPU stability and performance.

## Quick Start

### Prerequisites
- Windows 10/11
- Python 3.7+
- Install the benchmarks you want to test:
  - [3DMark](https://www.3dmark.com/)
  - [Unigine Superposition/Heaven/Valley](https://benchmark.unigine.com/)
  - [UL Procyon](https://benchmarks.ul.com/procyon)

### Installation
1. Clone this repository
2. git clone https://github.com/yourusername/GFX-API-Script.git
3. cd GFX-API-Script
4. Double-click `run.bat`

### Usage
1. Run `run.bat`
2. Select benchmarks from the GUI
3. Set test duration (in minutes)
4. Enter your board number
5. Configure benchmark settings if needed
6. Click Submit

Results are automatically saved to timestamped folders with complete logs and timing data.
